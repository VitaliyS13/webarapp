# Мультимаркерна WebAR Система

Вебзастосунок доповненої реальності (WebAR) для одночасного відстеження кількох маркерів і взаємодії з тривимірними об'єктами в реальному часі через браузер. Проєкт реалізовано в межах індивідуального завдання з виробничої практики.

Система є повністю клієнтською (Client-Side), кросплатформною та не потребує встановлення сторонніх додатків на мобільні пристрої — робота відбувається безпосередньо через веббраузер з підтримкою WebGL.

## 🚀 Технологічний стек

* **HTML5 / CSS3** — структура та адаптивний інтерфейс користувача.
* **JavaScript (ES6+)** — модульна бізнес-логіка застосунку.
* **A-Frame (v1.5.0)** — вебфреймворк для декларативного керування 3D-сценою (Entity-Component-System).
* **AR.js** — бібліотека комп'ютерного зору для відстеження маркерів та обчислення матриць трансформацій.
* **Three.js / WebGL** — низькорівневий рендеринг тривимірної графіки з апаратним прискоренням.

## 🌐 Автономне розгортання (Production)

Для повної автономності застосунок розгорнуто на безкоштовному хостингу статичних сайтів **GitHub Pages** (або Netlify). Завдяки автоматичній генерації SSL-сертифіката та підтримці протоколу `https://`, мобільні браузери надають безперешкодний доступ до камери на будь-яких пристроях без додаткових конфігурацій.
